## üîß Troubleshooting

### **Issue: Animation not showing**

**Check**:
1. ‚úÖ File exists in `assets/lottie/`
2. ‚úÖ File name matches exactly (case-sensitive)
3. ‚úÖ pubspec.yaml includes: `- assets/lottie/`
4. ‚úÖ Run `flutter pub get` after adding files
5. ‚úÖ Restart app (hot reload might not pick up assets)

### **Issue: App crashes when loading animation**

**Solutions**:
1. Check JSON file is valid (open in text editor)
2. Try a different animation file
3. Ensure file size is reasonable (<200KB)

### **Issue: Animation is too big/small**

**Adjust size in code**:
